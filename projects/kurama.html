    • Kurama - 9 Bit FPGA CPU

    Project Description

    This is a project I worked on in CSE 141L (a course I took in UCSD). It was particularly interesting since we were restricted to using 9-bit instruction words (compare that to the ByteFrost's 16 bits) and since it was application-specific, made to run 3 specific programs. I designed a 9-bit ISA and designed and implemented Kurama using SystemVerilog. To avoid writing code in machine language, I wrote an assembler with Java with label support and derived instructions for ease of programming.

    The name comes from Kurama, the nine-tailed demon fox from Naruto; each instruction bit thus can be said to represent one of Kurama's tails.
